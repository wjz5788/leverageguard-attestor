# LiqPass 前端应用路由接口总览

## 项目概述
LiqPass 是一个基于 React + TypeScript 的支付链接管理应用，提供支付链接创建、管理和支付功能。

## 路由架构

### 基础路由配置
```typescript
// App.tsx 中的路由配置
<Routes>
  <Route path="/" element={<Landing t={zh} />} />
  <Route path="/links" element={<Links t={zh} />} />
  <Route path="/links/create" element={<CreateLink t={zh} />} />
  <Route path="/pay/:id" element={<Payment t={zh} />} />
  <Route path="/success" element={<Success t={zh} />} />
</Routes>
```

## 页面路由详情

### 1. 首页 (Landing)
- **路径**: `/`
- **组件**: Landing
- **功能**: 支付链接生成、钱包连接、链接预览
- **文档**: [首页路由接口.md](./首页路由接口.md)

### 2. 支付链接列表 (Links)
- **路径**: `/links`
- **组件**: Links
- **功能**: 支付链接管理、状态筛选、搜索功能
- **文档**: [支付链接列表路由接口.md](./支付链接列表路由接口.md)

### 3. 创建支付链接 (CreateLink)
- **路径**: `/links/create`
- **组件**: CreateLink
- **功能**: 支付链接参数配置、链接生成
- **文档**: [创建支付链接路由接口.md](./创建支付链接路由接口.md)

### 4. 支付页面 (Payment)
- **路径**: `/pay/:id`
- **组件**: Payment
- **功能**: 订单确认、钱包支付、网络验证
- **文档**: [支付页面路由接口.md](./支付页面路由接口.md)

### 5. 支付成功页面 (Success)
- **路径**: `/success`
- **组件**: Success
- **功能**: 支付成功确认、订单信息展示
- **文档**: [支付成功页面路由接口.md](./支付成功页面路由接口.md)

## 技术架构

### 前端技术栈
- **框架**: React 18 + TypeScript
- **路由**: React Router 6
- **样式**: Tailwind CSS
- **状态管理**: React Context API
- **构建工具**: Vite

### 上下文提供者
```typescript
<WalletProvider>    // 钱包连接状态管理
<ToastProvider>     // 消息提示管理
```

## 国际化支持
所有页面组件都支持国际化，通过 `t` prop 传递翻译字典。

## 响应式设计
- 移动端优先设计
- 断点系统：sm (640px), md (768px), lg (1024px), xl (1280px)
- 自适应布局和组件

## 组件依赖关系

### 通用UI组件
- `Card` - 卡片容器
- `Button` - 按钮组件
- `Badge` - 徽章组件
- `Input` - 输入框组件
- `Breadcrumb` - 面包屑导航

### 布局组件
- `Header` - 页面头部
- `Footer` - 页面底部

## 数据流架构

### 状态管理
- **本地状态**: React useState/useReducer
- **全局状态**: Context API (WalletContext, ToastContext)
- **路由状态**: React Router useParams/useLocation

### 数据流向
1. 用户操作触发状态变化
2. 状态变化驱动UI更新
3. 路由变化管理页面导航
4. 上下文提供者管理全局状态

## 开发规范

### 文件组织
```
src/
├── components/     # 可复用组件
├── pages/         # 页面组件
├── contexts/      # 上下文提供者
├── i18n/          # 国际化资源
├── types.ts       # 类型定义
├── utils.ts       # 工具函数
└── constants.ts   # 常量定义
```

### 命名约定
- 组件文件: PascalCase (如 `Landing.tsx`)
- 工具函数: camelCase (如 `buildLink`)
- 类型定义: PascalCase (如 `PaymentLink`)
- 路由路径: kebab-case (如 `/links/create`)
