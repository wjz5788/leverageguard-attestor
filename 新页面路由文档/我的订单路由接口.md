# 我的订单路由接口文档

## 基本信息
- **页面名称**: 我的订单 (OrdersPage)
- **路由路径**: `/account/orders`
- **组件文件**: OrdersPage 组件
- **页面类型**: 用户管理页面
- **访问权限**: 已登录用户
- **实现状态**: ❌ 待开发

## 路由配置
```typescript
// 需要添加到App.tsx的路由配置中
<Route path="/account/orders" element={<OrdersPage />} />
```

## 页面功能
- 显示用户订单列表
- 提供订单详情查看功能
- 支持订单状态管理

## 接口定义

### 组件接口
```typescript
function OrdersPage(): JSX.Element
```

### 数据接口
```typescript
interface Order {
  id: string;
  sku: string;
  amount: number;
  status: string;
}
```

### 依赖组件
- `Crumb`: 面包屑导航组件
- `DataTable`: 数据表格组件
- `Link`: React Router 链接组件

## 页面布局
```typescript
<>
  <Crumb items={[{to:"/",label:"首页"},{label:"我的订单"}]}/>
  <section className="mx-auto max-w-7xl px-4 py-12">
    <DataTable 
      rows={rows} 
      cols={cols} 
      actions={(r)=> <Link className="text-amber-700 hover:underline" to={`/account/orders/${r.id}`}>详情</Link>} 
    />
  </section>
</>
```

## 表格配置

### 列定义
```typescript
const cols: Col<typeof rows[number]>[] = [
  {key: "id", label: "ID"},
  {key: "sku", label: "SKU"},
  {key: "amount", label: "金额"},
  {key: "status", label: "状态"}
];
```

### 示例数据
```typescript
const rows = [
  {id: "ord01", sku: "24h", amount: 20, status: "paid"}
];
```

## 国际化文本
- 页面标题: "我的订单"
- 面包屑: 首页 → 我的订单
- 表格列名: ID, SKU, 金额, 状态
- 操作按钮: "详情"

## 响应式设计
- 使用 `max-w-7xl` 容器限制最大宽度
- 响应式内边距: `px-4 py-12`
- 数据表格支持水平滚动

## 技术特点
- 使用 `DataTable` 组件展示订单数据
- 支持行级操作（查看详情）
- 静态示例数据，可扩展为动态数据

## 路由跳转
- 订单详情页: `/account/orders/:id`
- 使用动态路由参数传递订单ID

## 注意事项
- 当前使用静态示例数据
- 可扩展为从API获取真实订单数据
- 支持订单状态筛选和搜索功能
