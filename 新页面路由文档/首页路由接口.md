# 首页路由接口文档

## 基本信息
- **页面名称**: 首页 (Landing)
- **路由路径**: `/`
- **组件文件**: `/src/pages/Landing.tsx`
- **页面类型**: 主页面

## 路由配置
```typescript
<Route path="/" element={<Landing t={zh} />} />
```

## 页面功能
- 支付链接快速生成和预览
- 钱包连接状态显示
- 跳转到高级创建页面
- 复制支付链接功能

## 接口定义

### Props 接口
```typescript
interface LandingProps {
  t: Dictionary;
}
```

### 组件状态
- `linkUrl`: string - 生成的支付链接URL
- `message`: string - 显示消息

### 主要方法
1. `createPaymentLink()` - 创建支付链接
2. `copyToClipboard()` - 复制链接到剪贴板

## 依赖组件
- `Card` - 卡片组件
- `Button` - 按钮组件
- `Badge` - 徽章组件
- `Link` - React Router 导航组件

## 上下文依赖
- `WalletContext` - 钱包连接状态
- `ToastContext` - 消息提示

## 国际化文本
使用 `zh` 字典中的以下字段：
- `hero.badge` - 页面徽章文本
- `hero.title` - 页面标题
- `hero.subtitle` - 页面副标题
- `linkCreated` - 链接创建成功消息
- `copied` - 复制成功消息

## 页面布局
- **左侧**: 标题、描述、操作按钮
- **右侧**: 支付链接预览区域

## 响应式设计
- 移动端: 单列布局
- 桌面端: 双列布局 (md:grid-cols-2)

## 注意事项
- 需要钱包连接才能生成支付链接
- 支付链接预览仅在生成后显示
- 支持Base网络检测
