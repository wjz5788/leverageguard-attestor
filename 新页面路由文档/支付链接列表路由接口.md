# 支付链接列表路由接口文档

## 基本信息
- **页面名称**: 支付链接列表 (Links)
- **路由路径**: `/links`
- **组件文件**: `/src/pages/Links.tsx`
- **页面类型**: 列表页面

## 路由配置
```typescript
<Route path="/links" element={<Links t={zh} />} />
```

## 页面功能
- 显示用户创建的支付链接列表
- 支付链接状态管理（活跃/过期）
- 创建新支付链接
- 查看支付链接详情
- 复制支付链接

## 接口定义

### Props 接口
```typescript
interface LinksProps {
  t: Dictionary;
}
```

### 组件状态
- `links`: Array - 支付链接列表数据
- `selectedLink`: Object - 选中的支付链接

### 主要方法
1. `handleCreateLink()` - 跳转到创建页面
2. `handleViewLink(link)` - 查看链接详情
3. `handleCopyLink(link)` - 复制链接
4. `handleDeleteLink(link)` - 删除链接

## 依赖组件
- `Card` - 卡片组件
- `Button` - 按钮组件
- `Badge` - 徽章组件
- `DataTable` - 数据表格组件
- `Breadcrumb` - 面包屑导航

## 上下文依赖
- `WalletContext` - 钱包连接状态
- `ToastContext` - 消息提示

## 国际化文本
使用 `zh` 字典中的以下字段：
- `links.title` - 页面标题
- `links.createButton` - 创建按钮文本
- `links.table.headers` - 表格列标题
- `links.status.active` - 活跃状态文本
- `links.status.expired` - 过期状态文本

## 页面布局
- **顶部**: 面包屑导航和创建按钮
- **主体**: 支付链接数据表格
- **表格列**: 名称、金额、状态、创建时间、操作

## 数据字段
```typescript
interface PaymentLink {
  id: string;
  name: string;
  amount: number;
  currency: string;
  status: 'active' | 'expired';
  createdAt: string;
  url: string;
}
```

## 响应式设计
- 移动端: 表格垂直滚动
- 桌面端: 完整表格显示

## 注意事项
- 需要钱包连接才能查看链接列表
- 支持链接状态筛选
- 提供批量操作功能
