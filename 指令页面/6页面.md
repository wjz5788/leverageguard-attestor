# 页面6：数据库设计

## 核心数据表
1. users - 用户表
2. roles - 角色表
3. permissions - 权限表
4. data_records - 数据记录表

## 表结构示例
### users表
| 字段名 | 类型 | 描述 |
|--------|------|------|
| id | INTEGER | 主键 |
| username | VARCHAR(50) | 用户名 |
| email | VARCHAR(100) | 邮箱地址 |
| created_at | TIMESTAMP | 创建时间 |

## 关系图
users 1--* data_records
users *--* roles
roles *--* permissions
