# 页面5：API接口文档

## 认证接口
- POST /api/auth/login - 用户登录
- POST /api/auth/logout - 用户登出
- GET /api/auth/profile - 获取用户信息

## 数据接口
- GET /api/data/list - 获取数据列表
- POST /api/data/create - 创建新数据
- PUT /api/data/update/{id} - 更新数据
- DELETE /api/data/delete/{id} - 删除数据

## 错误码说明
- 200: 请求成功
- 401: 未授权访问
- 404: 资源未找到
- 500: 服务器内部错误
