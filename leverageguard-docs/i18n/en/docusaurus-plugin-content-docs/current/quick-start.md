This page explains how to use LiqPass: purchase, verify API, submit claims, and view on-chain records.

## User Journey
1. Connect wallet (Base).
2. Select principal and leverage on the "Product Page", complete USDC payment.
3. Bind exchange read-only API in "API Settings" (OKX / Binance).
4. Select the order that needs to be claimed in "Orders", enter "Initiate Claim".
5. Click "Verify" to get evidence summary and estimated payout.
6. Click "Payout" on-chain, the contract will transfer USDC to your wallet.
