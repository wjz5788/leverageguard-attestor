# 常见问题解答 (FAQ)

## 概述

本文档收集了LiqPass用户最常见的问题和解答，帮助您快速解决使用过程中遇到的问题。

## 产品相关

### Q1: LiqPass是什么？

**A:** LiqPass是一个去中心化的保险平台，专门为加密货币交易者提供杠杆交易保险服务。当用户的仓位被强制平仓时，LiqPass会根据保险条款提供赔付。

### Q2: LiqPass与传统保险有什么区别？

**A:** 主要区别包括：
- **去中心化**: 基于区块链技术，无需中心化机构
- **透明性**: 所有交易和赔付记录在区块链上公开可查
- **自动化**: 智能合约自动执行赔付流程
- **全球化**: 面向全球用户，无地域限制

### Q3: LiqPass支持哪些交易所？

**A:** 目前支持的主要交易所包括：
- Binance
- OKX  
- Bybit
- 其他主流交易所（持续增加中）

### Q4: 保险费用如何计算？

**A:** 保险费用基于以下因素计算：
- 保险金额
- 杠杆倍数
- 保险期限
- 市场波动率
- 用户风险评分

计算公式：`保费 = 保险金额 × 基础费率 × 杠杆系数 × 期限系数 × 风险系数`

## 使用相关

### Q9: 如何购买保险？

**A:** 购买流程：
1. 连接钱包到LiqPass平台
2. 选择保险参数（金额、杠杆、期限）
3. 确认保费并支付
4. 等待交易确认
5. 保险生效

### Q10: 如何申请赔付？

**A:** 赔付申请流程：
1. 登录LiqPass平台
2. 找到需要赔付的订单
3. 提交赔付申请
4. 提供清算事件证明
5. 等待审核和赔付

### Q11: 赔付审核需要多长时间？

**A:** 标准审核流程：
- **自动验证**: 1-2小时
- **人工审核**（如需要）: 24-48小时
- **赔付处理**: 1-2小时

总计通常在6-72小时内完成。

### Q12: 保险可以取消或退款吗？

**A:** 保险政策：
- **生效前**: 可以取消并获得全额退款
- **生效后**: 不可取消，保费不予退还
- **特殊情况**: 联系客服处理

## 资金安全

### Q13: 资金是否安全？

**A:** 我们采取多重安全措施：
- **智能合约审计**: 经过多家知名安全公司审计
- **资金隔离**: 用户资金与运营资金分离
- **多重签名**: 关键操作需要多重签名确认
- **保险基金**: 设立专门的保险基金

### Q14: 私钥和助记词如何保管？

**A:** 重要安全建议：
- ✅ **务必**: 备份助记词并安全存放
- ✅ **务必**: 使用硬件钱包存储大额资产
- ❌ **切勿**: 截图或拍照保存助记词
- ❌ **切勿**: 将助记词发送给任何人
- ❌ **切勿**: 使用在线工具生成助记词

### Q15: 如果忘记密码或丢失私钥怎么办？

**A:** 处理方案：
1. **忘记密码**: 可以通过助记词恢复钱包
2. **丢失私钥**: 使用备份的助记词恢复
3. **两者都丢失**: 无法恢复，资产将永久丢失

**重要**: 请务必妥善保管助记词！

## 费用相关

### Q16: 有哪些费用？

**A:** 主要费用包括：
- **保险费**: 购买保险的主要费用
- **网络Gas费**: 区块链交易费用
- **服务费**: 平台运营费用（已包含在保费中）

### Q17: Gas费为什么有时很高？

**A:** Gas费影响因素：
- **网络拥堵**: 以太坊网络繁忙时费用上涨
- **交易复杂度**: 复杂操作需要更多Gas
- **时间选择**: 不同时段费用差异较大

**建议**: 使用Polygon或Arbitrum网络以节省Gas费。

### Q18: 如何降低使用成本？

**A:** 成本优化建议：
1. 选择Polygon或Arbitrum网络
2. 在网络不拥堵时操作
3. 合理设置Gas价格
4. 批量操作减少交易次数

## 故障排除

### Q19: 交易一直处于等待状态怎么办？

**A:** 解决方案：
1. **检查网络状态**: 确认网络正常运行
2. **增加Gas费**: 适当提高Gas价格
3. **取消交易**: 在钱包中取消卡住的交易
4. **联系支持**: 如果问题持续存在

### Q20: 连接钱包失败怎么办？

**A:** 排查步骤：
1. 确认使用支持的钱包
2. 检查网络连接
3. 尝试刷新页面
4. 清除浏览器缓存
5. 联系客服获取帮助

### Q21: 赔付申请被拒绝的原因？

**A:** 常见拒绝原因：
- 提供的证据不足或不清晰
- 清算事件不在保险有效期内
- 赔付金额超过保险限额
- 违反保险条款

## 法律合规

### Q22: LiqPass是否受监管？

**A:** 监管状态：
- LiqPass作为去中心化协议运营
- 在某些司法管辖区可能受到相关法规约束
- 用户需自行了解当地法律法规
- 建议咨询专业法律顾问

### Q23: 税务如何处理？

**A:** 税务注意事项：
- 保险赔付可能产生税务义务
- 不同国家/地区税务政策不同
- 建议保留所有交易记录
- 咨询专业税务顾问
