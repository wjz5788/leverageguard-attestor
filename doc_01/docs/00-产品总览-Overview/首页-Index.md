## 问题陈述

在去中心化金融（DeFi）领域，用户面临清算风险时缺乏有效的保险机制。传统保险产品无法满足DeFi场景的实时性和透明性需求。

## 价值主张

LiqPass提供：
- **实时保护**：针对DeFi清算事件的即时保险
- **透明可信**：基于智能合约的自动化赔付
- **成本可控**：按需购买，杠杆可调
- **风险隔离**：资金与逻辑分离，降低系统性风险

## 核心功能

- **保险购买**：用户可配置杠杆倍数和保险期限
- **自动赔付**：检测到清算事件后自动触发赔付
- **风险控制**：多层验证机制确保赔付准确性
- **透明审计**：所有操作链上可查，完全透明
